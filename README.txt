Данный проект "Сайт для кадрового агенства" является учебным, использовался для изучения платформы ASP.NET и архитектуры MVC. 
Соединение с базой данных MsSQL реализовано с помощью фреймворка EntityFramework, для вёрстки использовалась библиотека Bootstrap 5.
Сайт хранит вакансии и резюме. Резюме можно добавлять в избранное, список избранных можно посмотреть отдельно. 
Также с помощью формы с валидацией можно добавить как вакансию, так и резюме.

Проект не доработан и падает при попытке отправить форму регистрации пользователя. Дело в том, что возникли проблемы с фреймворком
Identity и системной авторизацией C# - не хватает знаний для грамотной реализации.
